# test_render
Analizamos un catálogo de vehículos en 2 gráficas: 

1. Un histograma del odómetro de los vehículos que se observará al seleccionar la casilla de verificación 'Construir un histograma'.
2. La dispersión del odómetro vs el precio que se observará dando clic al botón 'Observa la dispersión de km recorridos vs precio'

Link de dashboard: https://test-render-30gx.onrender.com
